## Run locally

This is a static website. You can open `index.html` directly in a browser, or run a local static server.

- Quick (Python built-in server):

```powershell
cd "d:\\my-first-project"
python -m http.server 8000
```

- With Node/npm (uses `npx serve` via the provided script):

```powershell
cd "d:\\my-first-project"
npm start
```

- Or open `index.html` with VS Code Live Server extension.

If you want, I can add a full `npm install` flow or install `serve` as a dev dependency.
