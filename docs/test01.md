# 정의
클로드와의 협업을 통한 프로젝트 개발 테스트 ver01

# 주제
1. 개발자는 1인이다.
2. 개발자는 기획서를 제공한다.
3. 클로드는 기획서를 읽고 프로그램 진행 안내와 코드를 생성한다.

# 1차 개발 목표
1. git 환경 구축 과 정상적인 설치까지 테스트 완료

# 2차 개발목표 
1. 로컬pc에서 테스트 홈페이지 제작

# 3차 개발목표
1. 테스트 홈페이지에 다음과 같이 출력
   1.  출력 : 안녕하세요. 테스트 홈페이지 입니다.
   2.  ID : 16자리 입력칸
   3.  PW : 16자리 입력칸
   4.  회원가입
       1.  링크페이지 생성
       2.  id : 
       3.  pw :
       4.  생성하기
    5. 비밀번호 찾기
       1. id /pw 출력.
       2. pw 변경 링크 출력
          1. new pw :
